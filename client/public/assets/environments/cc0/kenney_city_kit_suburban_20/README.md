# Kenney City Kit Suburban + Roads (CC0)

This directory combines CC0 assets from two Kenney packs for the suburban map:

- **City Kit (Suburban)** – `kenney_city-kit-suburban_20.zip`
- **City Kit (Roads)** – `kenney_city-kit-roads.zip`

The GLB assets and textures are stored under `glb/`, with road meshes in `glb/roads/`.
